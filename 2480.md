# 2480
주사위 3개를 던져서  
 * 모두 같으면 : 10000 + (주사위 눈)*1000
 * 두 개만 같으면 : 1000 + (주사위 눈)*100
 * 모두 다르면 : 100*(가장 큰 주사위 눈)
위 규칙을 따르는 게임을 한다.  
***
입력 : 세 수를 입력 받음
***
## 알고리즘
세 가지 경우에 해당하는 경우에 따라, 더하는 수가 적용되도록 한다.
```cpp
#include <iostream>

using namespace std;
int main(){
    int a,b,c;
    cin >> a >> b >> c;
    cout << 1000*(a+10)*(a==b && b==c) + 100*((a+10)*(a==b)+(b+10)*(b==c)+(c+10)*(c==a))*(!(a==b && b==c)) + max(a,max(b,c))*100*(a!=b && b!=c && a!=c) << endl;
    return 0;
}
```
